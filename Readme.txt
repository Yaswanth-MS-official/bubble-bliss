🧋 Bubble Bliss: Bubble Tea Shop Website

Bubble Bliss is a modern Bubble Tea Shop Booking and Ordering Website built using HTML, CSS, Bootstrap 5, and JavaScript.

✨ Features
📱 Responsive Layout

Fully optimized for mobile, tablet, and desktop screens.

🍹 Menu Showcase

Displays various bubble tea options with high-quality images.

🧋 Drink Customization

A dedicated section allowing users to customize their drinks.

🎨 Modern UI

Built with Bootstrap 5 for a clean, professional, and modern design.

⚡ Interactive Components

Includes custom JavaScript for:

Navigation handling
Page interactivity
UI enhancements
